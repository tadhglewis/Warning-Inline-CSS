Warning-Inline-CSS

Extremely lightweight inline CSS detector.

Because inline CSS sucks

Never do this

<p style="background-color: red; color: white;">Never do this</p>

Otherwise you'll get this in your Console

Warning-Inline-CSS: Inline CSS detected...

And a nice little message containing the element so you can remove the inline CSS

Installation

GitHub

Download the warninginlinecss.js file and add the following right before the closing </body> tag

<script src="warninginlinecss.js"></script>
